Chapitre 2 

CLI : Command Line Interface

Types de connection:
- Console : mode hors ligne direct sur la machine
- SSH : En ligne connecté au serveur
- Telnet : SSH pas sécurisé (mdp en clair)
- AUX : connexion via cable téléphonique

Modes de commande :
- Mode d'exécution simple (toto>)
- Mode d'exécution privilégiée (root#)
